# Day 1 – Basics

## Linux Architecture
- **Kernel (K):** Written in C, talks to hardware  
- **Shell (S):** Talks to kernel  
- **Application (A):** User programs  

## Shell Types
- `sh` → /bin/sh  
- `bash` → /bin/bash (most used)  
- `csh`, `ksh`, `fish`
